# PINN-heat-equation
# 基于物理信息神经网络（PINN）求解热传导方程

使用 PyTorch 实现 Physics-Informed Neural Network，求解一维热传导方程，
并与解析解对比验证精度。

## 问题设定
- 方程：∂u/∂t = α·∂²u/∂x²，α=0.1
- 初始条件：u(x,0) = sin(πx)
- 边界条件：u(0,t) = u(1,t) = 0

## 方法
- 全连接网络（4层，Tanh激活），输入坐标(x,t)，输出温度u
- 损失函数 = 方程残差损失 + 初始条件损失 + 边界条件损失
- 通过自动微分（autograd）计算方程中的偏导数项

## 结果
- 最大绝对误差：0.0023
- 平均绝对误差：0.0005
- PINN预测与解析解高度吻合

## 技术栈
PyTorch · 自动微分 · 偏微分方程数值求解
